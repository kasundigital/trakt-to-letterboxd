Step 1 of 5
🎬 Connect Trakt
Enter your Trakt API credentials. These are needed to fetch your watch history.
Get your API credentials:
1. Go to trakt.tv/oauth/applications
2. Click New Application
3. Name it anything, set redirect URI to urn:ietf:wg:oauth:2.0:oob
4. Copy Client ID and Secret below
Private profile only
Only needed if your Trakt profile is private. See README for how to get this.
Step 2 of 5
🎞 Connect Letterboxd
Enter your Letterboxd credentials to enable automatic CSV upload after each sync.
How it works: After fetching from Trakt, the app logs into Letterboxd using a headless browser and uploads your CSV automatically. Your password is stored locally on your server only.
Stored locally in logs/config.json — never sent anywhere else.
Options
Enable Auto-Import
Automatically upload to Letterboxd after each sync
Step 3 of 5
⏰ Set Schedule
Choose when the app should automatically sync your Trakt history each day.
The app will sync automatically at this time every day. Pick a time when you're usually asleep.
What to sync
Watch History

Ratings

Watchlist
Include your Trakt watchlist in the CSV

📱 Telegram Notifications
Get a Telegram message after every sync with the film count and status. Totally optional.

1. Message @BotFather on Telegram → /newbot → copy token
2. Message @userinfobot to get your Chat ID
